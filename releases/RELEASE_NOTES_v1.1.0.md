# Foosboard v1.1.0

## Highlights

- Mantine-based table configuration workflow with drawer, stepper, uploads, previews, save and JSON download
- JSON-first Ullrich default table on startup
- Improved live board rendering so the startup view matches the configuration preview much more closely
- Stronger smoke and browser tests for startup rendering and preview stability
- Component split for better maintainability of the board, menu and configuration form
